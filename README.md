# ChatSLM

A free, open-source browser-based AI chatbot powered by SmolLM2 and running on Transformers.js

## Usage

ChatSLM is under active development. Come back later for when a live preview is available!

### Installation

#### 1. Clone Git repository

```sh
git clone https://github.com/natebabyak/smollm2-chatbot.git
cd smollm2-chatbot
```

#### 2. Install dependencies

```sh
# pnpm
pnpm install

# npm
npm install

# yarn
yarn install

# bun
bun install
```

#### 3. Start local development server

```sh
npm run dev
```
