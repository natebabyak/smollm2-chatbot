# ChatSLM

A free, open-source browser-based AI chatbot powered by SmolLM2 and running on Transformers.js

## Usage

A version of ChatSLM is available [here](smollm2-chatbot.vercel.app). 

### Installation

#### 1. Clone Git repository

```sh
git clone https://github.com/natebabyak/smollm2-chatbot.git
cd smollm2-chatbot
```

#### 2. Install dependencies

```sh
# pnpm
pnpm install

# npm
npm install

# yarn
yarn install

# bun
bun install
```

#### 3. Start local development server

```sh
npm run dev
```
